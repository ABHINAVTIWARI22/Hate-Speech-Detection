Hate Speech Detection
This project aims to detect "Hate Speech" from social media (Twitter) data.
Project Overview
Technologies used: Python, Pandas, NumPy.
Dataset: Uses the twitter.csv file.
Goal: To identify and classify text containing hate speech using machine learning techniques.
How to Run:--
Clone this repository or download the notebook file Hate_Speech_Detection_Project (1).ipynb.
Open the file in Google Colab.
Upload the twitter.csv file into the file section of the Google Colab session.
Run the code cells sequentially to process the data and see the results.
